근태관리 서비스 버전 업데이트 공지(3/18)

2021-03-18 | 작성자 : 운영자

안녕하세요. 아이브웍스 입니다. 근태관리 서비스를 보다 쉽고 편리하게 사용할 수 있도록 v1.4.0으로 업데이트 되었습니다. 자세한 내용은 아래 릴리즈 노트 내용을 참고 하여 주십시오. # 개선사항

  1. UI / UX 개선
    • UI / UX 개선을 통해 보다 쉬운 근로 정보 조회 및 관리 할 수 있도록 개선
  2. 회원가입
    • 비빌번호 초기화 페이지 추가
  3. 대시보드
    • 그래프 출력 시 인원 정보 표시
    • 일간 날짜 선택 이동 기능 추가
    • 소정근로, 연장근로 카드 정보 표시 수정
      • 기존 시간 표시 "08:00:00"를 "8시간"으로 표시 개선
    • 개인 대시보드 근로 유형에 따른 근로 정보 출력 개선
    • 관리자 및 부서장 대시보드 통합
    • 관리자 및 부서장 52시간 초과 근로 위험자 정보 카드 추가
    • 근로 일정 등 데이터가 없는 경우 알림 메시지 추가
  4. 근로템플릿관리
    • 자유출퇴근제 근로 유형 추가
  5. 근로스케줄관리
    • 근로 스케줄 정보 출력 순서 조정
      • 1차 회사 스케줄
      • 2차 부서 스케줄
      • 3차 사용자 스케줄
